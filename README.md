# ohjelmointiTA
Repositoriossa on ohjeet ja dokumentaatio sovelluskehittäjän perustyöasemasta. Wiki-sivuilta löydät dokumentaation asennuksesta sekä eri ohjelmistoihin tehdyistä asetuksista.
